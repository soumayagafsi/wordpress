== Description ==

This theme is perfectly and professionally built for call centers, BPO’s, contact center, telemarketing, telesales, online services and customer service websites. This theme is created with unique web design which is eye catching and attracts visitors. This theme has attractive design, style, colour, layout and functionality which are compatible and responsive with all devices.

If you are looking for an impressive theme with great attraction then don’t wait just go for it because this theme also have customization feature where you can edit colour, web layout, favorite images, and also does not required any technical expertise of handling this theme.



== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
